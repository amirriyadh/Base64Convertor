# Base64 Convertor

Online base64 convertor 

## Features
* Encode ```Text -> Base64```.
* Encode ```Base64 -> Text```.
* Show characters length for both text and base64 .
* UTF-8 Support.

## Built with

* [js-base64](https://github.com/dankogai/js-base64) - Conversion
* [Semantic UI](https://semantic-ui.com/) - Front-end styling
